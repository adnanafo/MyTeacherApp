# 📚 My Teacher App
This is a mobile app I built to teach my students how to code!

## 🛠 What is inside?
- **React Native:** The framework used to build the app.
- **Expo:** The tool that runs the app on our phones.
- **Termux:** The Linux system on my Android where the code lives.

## 🚀 How to use this project
If you are my student and want to try this code:
1. Install **Termux** on your phone.
2. Type `git clone https://github.com/your-username/MyTeacherApp.git`.
3. Type `npm install`.
4. Type `npx expo start`.
